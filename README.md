# Diabetes-ML-Classification
Projeto acadêmico de Machine Learning utilizando Python para aplicação de técnicas básicas de aprendizado de máquina e análise de dados.

# Diagnóstico de Diabetes com Machine Learning

Projeto acadêmico desenvolvido como parte dos estudos em Introdução a Inteligência Artificial durante a graduação em Engenharia de Computação na Universidade de Brasília.

O objetivo do projeto é aplicar técnicas de **Machine Learning** para prever a presença de diabetes a partir de dados clínicos.

## Objetivo

Desenvolver um modelo de classificação capaz de identificar se um paciente possui ou não diabetes com base em variáveis médicas presentes no dataset.

## Dataset

O projeto utiliza o **Pima Indians Diabetes Dataset**, que contém informações médicas de pacientes, incluindo:

- Número de gestações
- Nível de glicose
- Pressão sanguínea
- Índice de massa corporal (BMI)
- Idade
- Histórico familiar de diabetes

## Tecnologias utilizadas

- Python  
- PyTorch  
- Pandas  
- NumPy  
- Jupyter Notebook  

## Estrutura do projeto

O notebook contém todo o processo do projeto, incluindo:

- Preparação dos dados
- Treinamento do modelo
- Avaliação dos resultados
